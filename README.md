# 📚 Laboratório de Estudos em Bioinformática

Este repositório regista a minha progressão técnica, desde os fundamentos da lógica de programação até à manipulação avançada de dados genómicos.

### 📈 Jornada de Aprendizagem:
* **Níveis 1-7**: Domínio de sintaxe, estruturas de repetição e lógica fundamental.
* **Níveis 8-11**: Implementação de algoritmos para análise de sequências FASTA, processamento de dados de "pacientes" e geração de relatórios técnicos.
* **Infraestrutura**: Estudos de rede, profiling de código e depuração para alta performance.

### 🔬 Tópicos Praticados:
* Manipulação de ficheiros de sequências (DNA/Proteína).
* Lógica aplicada a problemas biológicos.
* Automação de relatórios de análise.

---
*Status: Em constante expansão conforme novos níveis de complexidade são atingidos.*
